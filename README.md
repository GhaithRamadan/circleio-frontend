# Circleio / Crypto Wallet UI 

Ein kleines Frontend Webprojekt das ich zur praktischen Anwendung meiner Grundlagen in **HTML, CSS und JavaScript** umgesetzt habe.

## Seiten
- Home (Landing Page mit Trading-UI)
- Wallet (Portfolio-Übersicht)
- Contact (Kontaktformular)

## Funktionen
- Mehrseitiges Layout mit Navigation
- Dark-/Light-Mode Umschaltung
- Responsives Design für verschiedene Bildschirmgrößen
- Einfache Interaktionen mit JavaScript

## Technologien
- HTML
- CSS
- JavaScript

## Ziel des Projekts
Ziel dieses Projekts war es, grundlegende Konzepte der Frontend-Entwicklung wie Layout, Styling, Responsivität und einfache Interaktivität praktisch umzusetzen.
